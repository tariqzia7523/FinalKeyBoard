Otobi Keyboard is the Bangla keyboard for Android in which you are skilled.

Features:
- Bijoy-like layout for Bangla 
- Adjustable keyboard height for more screen space
- Number row
- Swipe space to move pointer
- Delete swipe
- Ads-free
- Based on Android Open Source Project keyboard
- Free and open-source

Website: https://otobikb.github.io

This app is open-source, the code is available here: https://github.com/OtobiKB/OtobiKeyboard
